Scene Properties

 

Upon creating a new Scene, you will notice a small pop up window called Scene Properties will appear.

 

 

Name

The name of the scene

 

UID

The UID of the scene. It is empty if you create a new scene because it is calculated automatically on creation and cannot be edited. It shows the unique identifier of the backend scene-document which can be opened in your file explorer by clicking the Open button.

 

Open

Opens the backend scene-document in your file explorer.

 

Disable Live-Preview

Disables the Live-Preview display for that scene only.
