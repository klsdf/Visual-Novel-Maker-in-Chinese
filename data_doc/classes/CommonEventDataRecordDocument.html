CommonEventDataRecordDocument Class

Extends gs.DataRecordDocument
Module: gs

Stores a common-event defined in database.

Properties

items.data.commands

gs.SceneCommand

Contains all commands that will be executed by the common-event.

items.data.conditionEnabled

Boolean

Indicates if a switch needs to be set to ON before the common event will be trigger. That has no effect startCondition is set to "Auto".

items.data.conditionSwitch

gs.Variable

Switch to check if items.data.conditionEnabled is set to true.

items.data.parallel

Boolean

Indicates if the common-event runs in parallel.

items.data.parameter

gs.CommonEventParameter

Contains all parameter definitions of the common-event.

items.data.startCondition

Number

The common event's start condition.

  • 0 = Call - Common Event needs to be called by command.
  • 1 = Auto - Common Event runs automatically
